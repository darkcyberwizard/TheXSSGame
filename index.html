Game Information

  • The XSS Game is used to train players on Cross-site scripting attacks.
  • Accept the privacy policy and press Start Game to play.
  • Enter the player name and select an avatar.
  • Select a player role.
  • Select the question tiles at the bottom to answer the questions to complete the levels.

    Use a VPN

  • Remember, there could be multiple answers for a question.
  • Maximum possible points : 92 ( Correct answer: +2 points and Incorrect answer: -1 point)
  • You can enter the Simulator Zone to interact with elements by clicking the target icon, if it is visible.
  • You can always replay the game by selecting a different role.
  • You are awarded different badges for the of completed roles and simulation zone tasks.
  • You can view the player stats by clicking the stats button.
  • Click Exit to end the game.
  • Don't refresh or close the web browser as this will delete and reset your answers.

GDPR Privacy Notice for The XSS Game Players

  • We are dedicated to safeguarding the privacy of The XSS Game players by collecting non-personal data for improving the gaming experience.
  • The data is used internally for analysis, enhancing the game without sharing with third parties.
  • Strong security measures are in place to protect this data.
  • Players have rights under GDPR, including data access and rectification, which can be exercised by contacting nipuna.h.weeratunge@ntnu.no.
  • A more detailed Privacy Policy can be found here.
